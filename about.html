Finished product

Wyvern Chess

Wyvern Chess is a fantasy strategy game played across three stacked 12x8 boards: the Sky, the Land, and the Underworld.

  • Lessons with Grukk, Quick Match, Warlord Mode, and Pass & Play
  • Sky, Land, and Underworld pieces with layered tactics
  • Sprite art, move log, captured tray, hints, and tilt/flat board views
  • Playable directly in the browser through GitHub Pages

Foundry VTT modules

DM LM Pro and Downtime Master are finished Foundry VTT tools built to reduce live-session friction and keep campaign management moving between games.

💡

DM LM Pro

A GM-only lighting control centre for Foundry VTT. It puts scene mood, token lights, ambient light placement, blackout controls, memory slots, and export tools into one floating, draggable panel.

  • 11 scene presets and smooth fades
  • Token and ambient light presets with custom slots
  • Install in Foundry with Setup > Add-on Modules > Install Module, then paste the manifest URL from the README.

⏳

Downtime Master

A comprehensive downtime management system for D&D 5e in Foundry VTT. It replaces basic downtime handling with requests, crafting, training, resources, factions, harvesting, shops, treasure, and GM review tools.

  • Player request system, crafting, scribing, training, and history
  • 37+ factions, 90+ harvesting tables, and 799 blueprints
  • Install from Foundry's module search or paste the manifest URL from the README.
