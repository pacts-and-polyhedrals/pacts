Wyvern Chess

A fantasy strategy game played across three stacked 12x8 boards: Sky, Land, and Underworld. Learn with Grukk, play against the goblin AI, challenge a friend in Pass & Play, or use Teams Mode for table play.

  • Lessons, Quick Match, and Warlord Mode
  • Dragons, griffons, clerics, basilisk freezes, eruptions, and more
  • Move log, captured tray, hints, and tilt/flat board views

DM LM Pro

A GM-only lighting control centre for Foundry VTT. Manage scene mood, token lights, and ambient light placement from one floating, draggable panel during live play.

  • 11 scene presets with smooth fade support
  • Token and ambient light presets with custom slots
  • Install in Foundry with Setup > Add-on Modules > Install Module, then paste the manifest URL from the README.

⏳

Downtime Master

A comprehensive downtime management system for D&D 5e in Foundry VTT, built to keep players engaged between sessions while giving GMs review, sync, and world-management tools.

  • Requests, crafting, training, resources, and history
  • 37+ factions, 90+ harvesting tables, and 799 blueprints
  • Install from Foundry's module search or paste the manifest URL from the README.
